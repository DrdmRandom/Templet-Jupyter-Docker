This is a template for running Jupyter Notebook using a Docker container.  
It provides a ready-to-use environment for machine learning and deep learning projects, ensuring a smooth and consistent setup across different systems.

## Available Templates

There are two available templates in this setup:

- **TensorFlow** – Ideal for users who want to work with Google's TensorFlow framework for deep learning and AI applications.
- **PyTorch** – Suitable for users who prefer the PyTorch framework, widely used for research and production in deep learning.

You can choose the appropriate template depending on your system's compatibility and hardware capabilities.

## Prerequisites

Before using this template, ensure that you have the following installed on your system:

- **Docker** (latest version recommended)
- **NVIDIA Container Toolkit** (the version compatible with the model)
- **Python** (for additional scripting if needed)
